Rendering notes for Hammerspoon implementation:
  • All mockups render in hs.canvas — rectangles, rounded corners, text elements.
  • Translucent background with blur is tricky in canvas; likely fall back to solid dark gray at 92% opacity (very close visually).
  • Unnamed rows ("Space N") use reduced alpha + italic for visual distinction.
  • Font: system default (San Francisco). Tabular numerals for the index column.
  • Position: center-top of trigger monitor, ~80px below menubar.
  • Width: fit longest row; min 220px.
  • Timeout: 2s, no animation in v1 (adding fade requires canvas transform tricks).
